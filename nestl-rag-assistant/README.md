# RAG Assistant for Teacher FAQs using the NESTL Toolkit

This project is a static walkthrough of a Retrieval-Augmented Generation (RAG) pipeline built to answer teacher FAQs using the NESTL (Neurodivergent Education for Students, Teaching & Learning) Toolkit.

## Project Objective

The goal of this project is to demonstrate how a long-form educational PDF can be transformed into a question-answering assistant using document chunking, embeddings, retrieval, and LLM-based generation.

## Pipeline Overview

1. PDF ingestion and preprocessing  
2. Text chunking for retrieval  
3. Embedding generation  
4. Vector storage  
5. Hybrid retrieval  
6. Response generation with an LLM  
7. Evaluation using test cases  

## Use Case

This assistant is designed to help teachers and educators access relevant guidance from the NESTL Toolkit more efficiently and support neurodivergence-inclusive teaching practices.

## Tech Stack

- Python
- Jupyter Notebook
- LangChain
- Vector store / retrieval pipeline
- OpenAI embeddings
- LLM-based response generation
