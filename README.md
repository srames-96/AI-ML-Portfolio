# AI/ML Portfolio

This repository contains selected AI/ML projects demonstrating applied experience in building LLM-powered systems, including Retrieval-Augmented Generation (RAG) pipelines and document understanding workflows.

## 1. RAG Assistant for Teacher FAQs (NESTL Toolkit)

A Retrieval-Augmented Generation (RAG) pipeline built to answer teacher FAQs using the NESTL (Neurodivergent Education for Students, Teaching & Learning) Toolkit.

### Key Components
- PDF ingestion and chunking
- Embedding generation
- Vector database retrieval
- Hybrid retrieval pipeline
- LLM-based answer generation
- Evaluation using structured test cases

### Folder
`nestl-rag-assistant/`

## 2. DocuVision: LLM-Based Document Extraction Pipeline

DocuVision is a document understanding pipeline that replaces traditional OCR-based systems using LLMs to extract structured data from unstructured financial documents.

### Key Features
- Preprocessing pipeline for document normalization
- LLM-based field extraction from unstructured inputs
- Postprocessing to convert outputs into structured JSON format
- Bounding box alignment for extracted fields
- Structured output generation for downstream systems

### Problem Solved
Traditional OCR systems struggle with variability in financial documents. This system improves flexibility and accuracy by using LLMs to understand document context and extract relevant fields dynamically.

### Folder
`docuvision-llm-extraction/`
